🇬🇧➡️🇫🇷 English to French Translation Project

📌 Overview

This project is an English to French translation system built using Natural Language Processing (NLP) techniques. It takes English text as input and generates the corresponding French translation.

---

🚀 Features

- Translate English text into French
- Simple and easy-to-use interface
- Supports sentence-level translation
- Can be extended for multiple languages
